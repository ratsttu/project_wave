Tutorial

Drag and connect the blocks together to create the desired outcome from the robot.
Press send to send the code to the robot. If the outcome that occured was not desired
then bring the robot back to the start and rearrange the blocks until the desired action occurs.
To send the code to the robot, Press ENTER.
